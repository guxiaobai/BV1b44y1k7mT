## 控制字符样式

- ASCII 码
- 字符样式

-- 

- 高四位表示背景色 | K | R | G | B |
- 低四位表示前景色 | I | R | G | B |

---

- K = 0 : 背景不闪烁
- K = 1 : 背景闪烁
- I = 0 : 深色字符
- I = 1 : 浅色字符

---

| Red | Green | Blue | Color   | 说明      |
| --- | ----- | ---- | ------- | --------- |
| 0   | 0     | 0    | Black   | 黑色      |
| 0   | 0     | 1    | Blue    | 蓝色      |
| 0   | 1     | 0    | Green   | 绿色      |
| 0   | 1     | 1    | Cyan    | 青色      |
| 1   | 0     | 0    | Red     | 红色      |
| 1   | 0     | 1    | Magenta | 洋红      |
| 1   | 1     | 0    | Yellow  | 黄色      |
| 1   | 1     | 1    | White   | 白色/灰色 |
